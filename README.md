Empty Teeworlds
=========

Maximum compressed server
---------------------------

Empty teeworlds is server mod where removed everything. You cant shoot, walk, jump, win. I (Zabidenhtf) made this just for fun because why not. Fully removed physics, controls, server events (you can only respawn or spectate), weapon system. 

Around 2500 lines of code removed.
